o calculo da media amostral é:
$$\bar x = \frac{1}{n}\sum_{i=1}^{n}(X_i)$$

o calculo da variãncia é dado por:
$$ var= \frac{1}{n-1} \sum_{i=1}^{n}(X_i - \bar x)^2$$
